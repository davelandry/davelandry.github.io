D3 Basics

What is D3?

D3 is a JavaScript library that allows for manipulating HTML page elements in JavaScript using complex data arrays.

It also provides a lot of nice JavaScript shortcuts.

Let's load it!

We can use the JavaScript file directly from their website: 

<script src="https://d3js.org/d3.v4.min.js"></script>

Selections

Similar to JavaScript and JQuery (if you are familiar), D3 has it's own element selector: 

document.getElementById("name");
$("#name");
d3.select("#name");

Attributes and Styles

Element attributes and styles can be changed by using the supplied D3 functions: 

var elem = d3.select("#name");

elem.attr("class", "big");
elem.style("color", "red");

Chaining Styles

The attr and style functions always return the original element being modified, so they can be chained one after another: 

d3.select("#name").attr("class", "big").style("color", "red");

Multi-Selection

You can also select multiple elements on a page and change all of their properties at the same time using the selectAll function: 

d3.selectAll(".big").style("color", "red");
