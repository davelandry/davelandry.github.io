Homework

Whether storing in a file on your computer or using an API, set up an HTML page that loads the data you need for your final project into Javascript using D3. Use console.log to display your data and we will look at it next class all together.

Don't worry about drawing anything based on the data, this exercise is solely for exploring the data in the console.

Homework Example

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <script src="https://d3js.org/d3.v7.min.js"></script>
  </head>
  <body>
    <script>
      d3.csv("pokemonStats.csv")
        .then(function(data) {
          console.log(data);
        });
    </script>
  </body>
</html>
